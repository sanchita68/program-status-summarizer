# Program Status Summarizer

A tool for summarizing program status reports using AI.

## Overview
This repository contains code and data for automated 
program status summarization.

## Files
- `summarizer.py` — Main summarization script
- `sample_data.csv` — Sample program data  
- `status_report.md` — Sample status report output
- `requirements.txt` — Python dependencies
